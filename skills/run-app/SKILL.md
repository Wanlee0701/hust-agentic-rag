# Run App Skill

Sử dụng skill này khi người dùng yêu cầu chạy thử ứng dụng Chatbot.

## Hướng dẫn thực thi:

1. Chạy lệnh sau trong terminal, bắt buộc ở thư mục gốc của dự án (`c:\Users\PC\Desktop\ĐATN`):
   `streamlit run app.py`

2. Báo lại cho người dùng URL Local (thường là `http://localhost:8501`) để người dùng có thể truy cập qua trình duyệt.
3. Không tự ý tắt task unless asked, vì UI cần duy trì server để người dùng sử dụng.
